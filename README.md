🎮 Hangman Game — CodeAlpha Internship Task 1
A simple text-based Hangman Game built with Python as part of the CodeAlpha Python Programming Internship.
📌 About the Project
The player tries to guess a hidden word one letter at a time.
Each wrong guess brings the hangman one step closer to completion.
Guess the word before 6 wrong attempts — or it's game over!
🚀 Features
5 predefined words chosen randomly each round
ASCII art hangman that updates with every wrong guess
Tracks correct and incorrect guesses
Replay option after each round
Clean console interface with emojis
